# Идентификация ботов среди пользовательских сессий
##  Цель:  
Разработать алгоритм, который автоматически распределяет пользователей по сегментам на основе их поведенческих признаков. Мы используем Spark ML для эффективного обучения и масштабируемости.  
## 🛠️ Как устроен пайплайн?
- Весь процесс разбит на два этапа.

  -  Обучение модели (PySparkMLFit.py)  
Загрузка данных: Используется файл session-stat.parquet — подготовленная таблица с признаками.  
Выбор алгоритмов: Проводится сравнение трех популярных моделей:  
Random Forest Classifier  
Logistic Regression  
Gradient Boosted Tree Classifier (GBT)  
Гиперпараметры: Для каждой модели настраиваем сетку параметров через CrossValidator.  
Результат: Лучшие параметры фиксируются в YAML-файле, а сама модель сохраняется в каталоге models/.

  -  Применение модели (PySparkMLPredict.py)  
Загрузка: Берется новая партия данных (например, test.parquet).    
Предсказание: Применяется лучшая модель из предыдущего этапа.    
Выход: Генерируется пара (session_id, predicted_segment) в формате Parquet.
